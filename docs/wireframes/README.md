# Serv.io Wireframes

Folder ini berisi *wireframes* kasar (lo-fi) untuk antarmuka proyek Serv.io. File-file ini berformat SVG (`.svg`) sehingga sangat ringan dan kompatibel dengan alat desain modern.

## Daftar Wireframe (Mid-Fidelity):

**Bagian Publik & Portal**
1. **`Tracking-Wireframe.svg`** - Halaman utama pelanggan untuk melacak tiket servis.
2. **`Login-Wireframe.svg`** - Halaman portal *login* untuk Staff (Admin/Teknisi).

**Bagian Admin**
3. **`Dashboard-Wireframe.svg`** - Halaman *overview* untuk Admin.
4. **`Pelanggan-Wireframe.svg`** - Halaman manajemen data pelanggan (*Customer List*).
5. **`Perangkat-Wireframe.svg`** - Halaman manajemen data perangkat (*Device List*).
6. **`Tiket-Servis-Wireframe.svg`** - Halaman detail tiket (form edit & log) di panel Admin/Staff.
7. **`Sparepart-Wireframe.svg`** - Halaman manajemen inventaris sparepart.
8. **`Invoice-Wireframe.svg`** - Halaman manajemen tagihan pelanggan.

**Bagian Teknisi**
9. **`Teknisi-Dashboard-Wireframe.svg`** - Halaman khusus Dashboard *Role* Teknisi (dengan sidebar yang dilimitasi khusus tugas Teknisi).
10. **`Teknisi-Tiket-Servis-Wireframe.svg`** - Halaman daftar tiket servis yang ditugaskan khusus ke teknisi yang sedang *login*.
11. **`Teknisi-Sparepart-Wireframe.svg`** - Halaman data inventaris dengan opsi *request* pengambilan *sparepart*.

## Cara Mengimpor ke Figma:
Figma mendukung import SVG secara *native*. Anda bisa melakukan salah satu dari dua cara berikut:

### Metode 1: Drag and Drop (Paling Mudah)
1. Buka aplikasi **Figma** (desktop atau *browser*).
2. Buat atau buka sebuah file *Design* (`New design file`).
3. Buka *File Explorer* (Windows) atau *Finder* (Mac) Anda.
4. Navigasikan ke folder proyek ini (`docs/wireframes/`).
5. Blok semua file `.svg`, lalu **tarik (drag)** dan **lepaskan (drop)** langsung ke atas kanvas Figma. 
6. Figma secara otomatis akan merender SVG tersebut sebagai *vector layers* yang bisa Anda ubah ukuran dan warnanya secara bebas!

### Metode 2: Import Menu
1. Buka file Figma Anda.
2. Klik logo Figma di pojok kiri atas -> **File** -> **Place image...** (atau tekan tombol `Ctrl + Shift + K` / `Cmd + Shift + K`).
3. Pilih file `.svg` dari folder `docs/wireframes/`.
4. Klik pada kanvas untuk menempatkan *wireframe*.

## Catatan untuk Desainer:
*   Karena *wireframe* ini berformat *vector* asli, setiap kotaknya (seperti tombol, form, teks *placeholder*) bisa diklik 2x (*double-click*) di Figma untuk diungroup dan diedit ulang.
*   Anda bisa mengubah struktur kasarnya menjadi rancangan UI (*High-Fidelity*) langsung dengan mewarnai dan menambahkan elemen Figma lainnya.

*(Anda juga bisa mengimpor file ini dengan cara yang persis sama ke aplikasi UI/UX lain seperti **Penpot**, **Sketch**, atau **Adobe XD**)*
